# Specification

## Summary
**Goal:** Replace slider controls with yes/no buttons in the daily check-in form.

**Planned changes:**
- Replace the slider control with two clickable buttons ("Yes" and "No") for each check-in question
- Add visual highlighting to indicate the selected button
- Ensure button controls are keyboard accessible and meet accessibility standards
- Apply the new button interface consistently across all check-in questions

**User-visible outcome:** Users can answer daily check-in questions by clicking clearly labeled "Yes" or "No" buttons instead of using a slider control.
